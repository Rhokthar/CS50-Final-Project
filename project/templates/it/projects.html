FINANCE

Un'applicazione web completa costruita per un Problem Set del Corso CS50 di Harvard. Usa l'API IEX per prendere in tempo reale il costo degli stock azionari, e funziona come portfolio degli stock acquistati. Back-end: Python, SQL - Front-End: Bootstrap. Implementerò un portafoglio reale.

SIMPLE FILTER APPLIER

Un altro programma costruito per un Problem Set del corso CS50 di Harvard. Programma Console programmato in C che applica filtri alle immagini, tramite ricalcolo del valore dei pixel ad uno ad uno.

PYTHON ARITHMETIC FORMATTER

Formattatore aritmetico costruito in Python per uno skill-test del corso di FreeCodeCamp "Scientific Computing with Python". Rappresenta addizioni e sottrazioni visualizzandole in colonna. Se l'opzione viene checkata, restituisce anche il risultato.

PYTHON TIME CALCULATOR

Calcolatore orario costruito in Python per un altro skill-test del corso di FreeCodeCamp "Scientific Computing with Python". Ricalcola l'orario dopo che l'utente aggiunge altro tempo ad un dato orario, tramite CLA. Mostra il nuovo orario in AM/PM e quanti giorni sono passati.

SQL FIFTYVILLE

Piccolo "gioco investigativo" fatto per il corso CS50 di Harvard. Dovetti interrogare alcuni DBs per cercare tracce di un furto in un negozio. Seguendo le briciole, trovai il colpevole e il suo (o sua 😉) complice.
