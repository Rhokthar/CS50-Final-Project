PYTHON ARITHMETIC FORMATTER

Arithmetic Formatter built in Python for a skill-test of FreeCodeCamp's Course "Scientific Computing with Python". It displays additions and substractions by visualizing them via column method. If checked, it also gives the result of the operation.

PYTHON TIME CALCULATOR
